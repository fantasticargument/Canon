Attention (Basic Printing)
NOTE
Install the printer driver.
The printer driver is software required for printing with this printer. Before printing, install the printer driver.
"Installing the Printer Driver"
When printing on custom size paper
You need to register the custom paper size in the printer driver previously.
"Registering a Custom Paper Size"
About the contents specified in "Basic Printing Procedure"
The specified settings apply only to data to be printed from the same application. Exiting the application resets the changed settings to the default settings.
For details on the procedure for changing the default values for printing preferences, see "Changing the Default Values for Printing Preferences."
