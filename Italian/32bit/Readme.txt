================================================================

1. Proporzioni

  È possibile che alcune applicazioni non stampino correttamente
  quando è stato impostata una proporzione di stampa. In questo 
  caso, impostare il valore del ridimensionamento su 100%.
  Se l'applicazione dispone di una propria funzione di 
  ridimensionamento, utilizzare tale funzione.

================================================================

2. Layout di pagina

  Alcune applicazioni permettono di stampare serie di copie 
  multiple. È tuttavia preferibile non utilizzare questa 
  funzione se nella scheda [Imposta pagina] di [Layout di 
  pagina] è selezionata un'opzione diversa da [1 su 1].
  Se questa funzione è in uso nell'applicazione, i dati 
  potrebbero non essere stampati in maniera corretta.

================================================================

3. Separatore di pagina

  In Windows non è possibile stampare il [Separatore di pagina].

================================================================

4. Autorizzazioni utente per l'installazione

  Per installare il driver della stampante, accedere al sistema 
  utilizzando un utente dotato di diritti amministrativi. Per 
  l'installazione è infatti necessaria l'autorizzazione 
  Controllo completo.

================================================================

5. Installazione di una stampante di rete e scaricamento del 
   software correlato (Windows 2000/XP/Server 2003)

  Quando si scarica il software e si installa una stampante di 
  rete in Windows 2000/XP/Server 2003, l'installazione può 
  essere eseguita utilizzando un utente non dotato di diritti 
  amministrativi.
  Se però la stampante di rete viene installata utilizzando un 
  utente non dotato di tali diritti, alcune funzioni possono non
  essere disponibili a causa di limitazioni dovute alle funzioni
  di sicurezza del sistema operativo.

  [Funzioni non disponibili]
  - Il programma di disinstallazione non viene registrato in 
    [Installazione applicazioni] ([Installazione applicazioni] 
    per Windows 2000).

================================================================

6. Installazione in un ambiente multiutente 
   (Windows XP/Vista/7/Server 2008)

  Se la stampante è installata in Windows XP/Vista/7/Server 
  2008, non installarla utilizzando un utente a cui si ha avuto 
  accesso con la funzione "Cambio rapido utente" di Windows 
  XP/Vista/7/Server 2008.
  Quando si utilizza la funzionalità "Cambio rapido utente", 
  accertarsi di eseguire nuovamente il logon prima 
  dell'installazione.

================================================================

7. Disinstallazione della stampante di rete collegata a un altro
   utente

  Anche se si esegue il programma di disinstallazione, le 
  informazioni relative alla stampante di rete presenti sul 
  computer locale di un altro utente non vengono eliminate, ma 
  restano nel suo sistema operativo come informazioni 
  particolari correlate all'utente.
  Se un altro utente ha installato la stampante sul proprio 
  computer locale tramite una rete, prima di eseguire il 
  programma di disinstallazione assicurarsi che l'utente elimini
  l'icona della stampante di rete dalla propria cartella 
  Stampanti.

================================================================

8. Limitazioni all'uso della stampante di rete

  Dopo aver installato una stampante di rete, non è possibile 
  utilizzarla come stampante locale modificando la destinazione 
  del collegamento.
  Per utilizzare la stampante come stampante locale, 
  disinstallare la stampante di rete installata, selezionare una
  porta locale e installare nuovamente il driver della 
  stampante.

================================================================

9. Riavvio di Windows con eventuali processi non stampati

  Se occorre riavviare Windows con un processo di stampa 
  residuo, mettere in pausa il processo di stampa dalla cartella
  della stampante e quindi riavviare Windows.

================================================================

10. La Modalità protetta di Internet Explorer 
    (Windows Vista/7/Server 2008)

  Se si utilizza Windows Vista/7/Server 2008 e si esegue 
  Internet Explorer facendo clic sull'icona relativa a 
  [Visualizza manuale] o altra voce dall'interno di 
  Installazione del CD-ROM, la [Modalità protetta] viene 
  impostata su [disattivata] indipendentemente dalle 
  impostazioni.
  Per impostare la [Modalità protetta] su [attivata], chiudere 
  Internet Explorer una volta e quindi riavviarlo manualmente.

================================================================

11. Caratteri per il nome della stampante 
    (Windows Vista/7/Server 2008)

  Se si utilizza il carattere "@" all'interno del nome della 
  stampante, questa non verrà riconosciuta dal sistema operativo
  a causa del nome non valido, pertanto le funzioni di stampa, 
  manutenzione e configurazione saranno disabilitate.

================================================================

12. Registrazione di formati carta personalizzati 
    (Windows Vista/7/Server 2008)

  Accedere al sistema utilizzando un utente dotato di diritti 
  amministrativi quando si apportano modifiche a [Formato carta 
  personalizzato] nella scheda [Imposta pagina] del driver della
  stampante.

================================================================

13. Autorizzazione di accesso a Proprietà stampante 
    (Windows Vista/7/Server 2008)

  Quando si accede al sistema utilizzando un account dotato di 
  diritti amministrativi diverso da quello utilizzato per 
  installare il driver della stampante, non sarà possibile 
  accedere alle proprietà della stampante con un utente dotato 
  di diritti amministrativi.
  Per accedere alle proprietà della stampante, eseguire una 
  delle procedure seguenti.

  - Windows Vista e Server 2008
    Fare clic con il tasto destro del mouse sull'icona della 
    stampante e scegliere [Esegui come amministratore].

  - Windows 7 e Server 2008 R2
    Selezionare [Modifica proprietà] nella scheda [Generale] 
    delle proprietà della stampante.

================================================================

14. Stampa di linee sottili (Windows Vista/7/Server 2008)

  In determinate applicazioni, lo spessore di una linea può 
  essere impostato a un valore inferiore a 1 punto, ma lo 
  spessore effettivo nella stampa potrebbe essere equivalente a 
  quello di una linea di 1 punto.

================================================================

15. Avvertenze da osservare quando si utilizza Microsoft Excel

  Quando si stampano dati di Microsoft Excel che includono celle
  combinate, può succedere che alcuni caratteri dei dati non 
  vengano stampati.

  Il fenomeno precedente dipende dalla modifica delle 
  impostazioni dello schermo nella finestra [Preferenze 
  (utenti)] o [Preferenze (amministratori)] della Finestra di 
  stato della stampante, pertanto è opportuno prestare 
  particolare attenzione quando si cambiano le impostazioni.

  - Impostazione di [Visualizza impostazioni temporizzazione] su
    [All'avvio del processo di stampa] (impostazione 
    predefinita: [In caso di errori])
  - Impostazione di [Ingrandisci finestra durante la 
    visualizzazione automatica] su Attivato (impostazione 
    predefinita: Attivato)

================================================================

16. Informazioni per gli utenti che effettuano l'aggiornamento 
    da Windows Vista a Windows 7

  Quando si aggiorna il sistema operativo da Windows Vista a 
  Windows 7, disinstallare prima il driver della stampante, 
  aggiornare il sistema operativo e quindi reinstallare il 
  driver della stampante quando è terminata l'operazione di 
  aggiornamento.
  Se si aggiorna il sistema operativo senza disinstallare il 
  driver della stampante, quest'ultimo potrebbe non funzionare 
  correttamente.
  In questo caso, disinstallare e poi reinstallare nuovamente il
  driver della stampante.

================================================================

17. Canon Driver Configuration Tool

  Questo driver supporta il programma Canon Driver 
  Configuration Tool versione 2.00 o successiva.
  È possibile modificare le impostazioni predefinite del driver 
  della stampante per creare impostazioni driver personalizzate 
  utilizzando il Canon Driver Configuration Tool.
  Per informazioni dettagliate su questa utility, consultare la 
  Guida dell'utente di Canon Driver Configuration Tool.
