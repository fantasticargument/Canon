1. Mise à l'échelle

  Il est possible que certaines applications n'impriment pas 
  correctement si une mise à l'échelle a été définie. Dans ce 
  cas, redéfinissez la mise à l'échelle sur 100 %.
  Si l'application possède une propre fonction de mise à 
  l'échelle, utilisez alors celle-ci.

================================================================

2. Mise en page

  Certaines applications disposent d'une fonction pour de 
  multiples jeux de copies. Evitez toutefois d'utiliser cette 
  fonction avec une configuration autre que [1 sur 1] définie 
  dans [Mise en page] de l'onglet [Configuration page].
  La disposition des données risque d'être incorrecte si cette 
  fonction est utilisée à partir de l'application.

================================================================

3. Page de séparation

  La [Page de séparation] sous Windows ne peut pas être 
  imprimée.

================================================================

4. Droits d'utilisateur lors de l'installation

  Ouvrez une session en tant qu'utilisateur ayant les droits 
  d'administrateur pour installer le pilote d'imprimante. La 
  permission de contrôle total est nécessaire pour 
  l'installation.

================================================================

5. Téléchargement et installation d'un pilote d'imprimante pour 
   une imprimante réseau (Windows 2000/XP/Server 2003)

  Lorsque vous téléchargez et installez un pilote d'imprimante 
  pour une imprimante réseau sous Windows 2000/XP/Server 2003, 
  vous pouvez l'installer en tant qu'utilisateur sans droits 
  d'administrateur.
  Cependant, si ce pilote d'imprimante pour l'imprimante réseau 
  est installé par un utilisateur ne disposant pas de droits 
  d'administrateur, certaines fonctions sont susceptibles de ne 
  pas être disponibles en raison de restrictions du dispositif 
  de sécurité du système d'exploitation.

  [Fonctions non disponibles]
  - Impossibilité d'enregistrer le programme de désinstallation 
    dans [Ajouter ou supprimer des programmes] 
    ([Ajout/Suppression de programmes] sous Windows 2000).

================================================================

6. Installation dans un environnement multi-utilisateur 
   (Windows XP/Vista/7/Server 2008)

  Lorsque l'imprimante est installée sous Windows 
  XP/Vista/7/Server 2008, ne l'installez pas pour un utilisateur
  changé par la fonction "Changement rapide d'utilisateur" de 
  Windows XP/Vista/7/Server 2008.
  Si vous utilisez la fonction "Changement rapide 
  d'utilisateur", veillez à rouvrir une session avant de 
  procéder à l'installation.

================================================================

7. Désinstallation de l'imprimante réseau raccordée sur un autre
   utilisateur

  Même si vous avez exécuté le programme de désinstallation, 
  l'information relative à l'imprimante réseau installée sur 
  l'ordinateur local d'un autre utilisateur restera dans son 
  système d'exploitation comme information propre à cet 
  utilisateur.
  Si un autre utilisateur a installé l'imprimante via le réseau 
  sur son ordinateur local, assurez-vous que cet utilisateur 
  supprime l'icône de l'imprimante réseau du dossier 
  d'imprimante avant d'exécuter le programme de désinstallation.

================================================================

8. Restriction sur l'imprimante réseau

  Lorsqu'une imprimante réseau a été installée, celle-ci ne peut
  pas être utilisée comme imprimante locale, même si vous lui 
  affectez un port local.
  Pour utiliser l'imprimante comme imprimante locale, 
  désinstallez l'imprimante réseau installée, sélectionnez un 
  port local, puis réinstallez le pilote d'imprimante.

================================================================

9. Redémarrage de Windows avec une tâche d'impression en attente

  Si vous devez redémarrer Windows lorsqu'une tâche d'impression
  est en attente, suspendez d'abord cette tâche à partir du 
  dossier d'imprimante. Redémarrez ensuite Windows.

================================================================

10. Le mode protégé d'Internet Explorer 
    (Windows Vista/7/Server 2008)

  Si vous utilisez Windows Vista/7/Server 2008 et que vous 
  exécutez Internet Explorer en cliquant sur l'icône pour 
  [Afficher le manuel] etc. dans la configuration du CD-ROM, 
  [Mode protégé] est configuré sur [désactivé] quels que soient 
  les paramètres.
  Si vous souhaitez mettre [Mode protégé] sur [activé], quittez 
  Internet Explorer, puis relancez-le manuellement.

================================================================

11. Caractères utilisés dans le nom de l'imprimante 
    (Windows Vista/7/Server 2008)

  L'utilisation du caractère "@" dans le nom de l'imprimante la 
  rend non reconnaissable par le système d'exploitation qui 
  considère que le nom d'imprimante est incorrect, ce qui 
  désactive l'impression, le fonctionnement et la configuration.

================================================================

12. Enregistrement du format papier personnalisé 
    (Windows Vista/7/Server 2008)

  Assurez-vous que vous êtes connecté comme utilisateur ayant 
  les droits d'administrateur pour modifier [Format papier 
  personnalisé] dans l'onglet [Configuration page] du pilote 
  d'imprimante.

================================================================

13. Droit d'accès aux propriétés de l'imprimante 
    (Windows Vista/7/Server 2008)

  Vous n'avez pas accès aux propriétés de l'imprimante comme 
  utilisateur ayant des droits d'administrateur si vous vous 
  connectez comme utilisateur administrateur mais que cet 
  utilisateur n'est pas celui qui a installé le pilote 
  d'imprimante.
  Pour accéder aux propriétés de l'imprimante, appliquez l'une 
  des procédures suivantes.

  - Windows Vista et Server 2008
    Cliquez avec le bouton droit sur l'icône de votre 
    imprimante, puis sélectionnez [Exécuter en tant 
    qu'administrateur].

  - Windows 7 et Server 2008 R2
    Cliquez sur [Modifier les propriétés] dans l'onglet 
    [Général] des propriétés de l'imprimante.

================================================================

14. Impression des lignes fines (Windows Vista/7/Server 2008)

  Certaines applications autorisent la définition de l'épaisseur
  d'une ligne d'une épaisseur inférieure à 1 point ; l'épaisseur
  réelle sur l'impression peut cependant être celle d'une ligne 
  d'une épaisseur de 1 point.

================================================================

15. Précautions à prendre pour l'utilisation de Microsoft Excel

  Lorsque vous imprimez des données Microsoft Excel contenant 
  des cellules fusionnées, certains caractères de ces données 
  risquent de ne pas s'imprimer.

  Comme le phénomène ci-dessus est provoqué par la modification 
  des paramètres de la boîte de dialogue [Préférences 
  (utilisateurs)] ou [Préférences (administrateurs)] de la 
  fenêtre d'état de l'imprimante (comme indiqué ci-dessous), 
  soyez prudent lorsque vous modifiez ces paramètres.

  - Paramétrage de [Afficher les paramètres de début et de fin] 
    sur [Au démarrage de l'impression] (valeur par défaut : [En 
    cas d'erreur])
  - Paramétrage de [Agrandir la fenêtre lors de l'affichage 
    automatique] sur Activé (valeur par défaut : Activé)

================================================================

16. Précautions à prendre pour la mise à niveau de Windows Vista
    vers Windows 7

  Lorsque vous mettez votre système d'exploitation à niveau de 
  Windows Vista vers Windows 7, désinstallez d'abord le pilote 
  d'imprimante, mettez à niveau le système d'exploitation, puis 
  réinstallez le pilote d'imprimante après cette mise à niveau.
  Si vous effectuez la mise à niveau du système d'exploitation 
  sans avoir désinstallé le pilote d'imprimante, ce dernier 
  risque de ne pas fonctionner correctement.
  Dans ce cas, désinstallez le pilote d'imprimante, puis 
  réinstallez-le.

================================================================

17. Canon Driver Configuration Tool

  Ce pilote prend en charge l'outil Canon Driver Configuration 
  Tool version 2.00 ou supérieure.
  Vous pouvez modifier les paramètres par défaut d'usine du 
  pilote d'imprimante de manière à créer des paramètres de 
  pilote personnalisés à l'aide de l'outil Canon Driver 
  Configuration Tool.
  Pour plus d'informations sur cet utilitaire, reportez-vous à 
  son guide de l'utilisateur.
