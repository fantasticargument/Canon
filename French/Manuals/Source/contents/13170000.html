 Erreur de service
<Cause> Condition anormale sur l'imprimante.
Solution
Appliquez la procédure décrite à la section "Lorsque le message <Erreur de service> s'affiche".
 Imprimante incompatible
<Cause> Vous ne pouvez pas utiliser l'imprimante connectée.
Solution
Connectez une imprimante prise en charge.
 Communication avec le serveur impossible
<Cause> L'ordinateur client ne parvient pas à communiquer avec le serveur d'impression.
Solution
Vérifiez l'état du serveur d'impression et de votre ordinateur.
 Erreur de communication
<Cause> Le câble USB n'est pas connecté ou l'imprimante est éteinte.
Solution 1
Branchez le câble USB.
Solution 2
Allumez l'imprimante.
 Erreur de carte réseau
<Cause> Le câble réseau (LAN) n'est pas connecté à l'adaptateur réseau. Autre cause possible : l'adaptateur réseau et l'imprimante sont éteints.
Solution 1
Branchez le câble réseau (LAN) sur l'adaptateur réseau.
Solution 2
Allumez l'imprimante et l'adaptateur réseau.
 Communication avec l'imprimante impossible
<Cause 1> Comme la fonction de communication bidirectionnelle n'est pas activée, l'ordinateur ne peut pas communiquer avec l'imprimante.
Solution
Activez la communication bidirectionnelle, puis redémarrez l'ordinateur et l'imprimante.
"Fonctionnement de base/Informations diverses pour Windows"
<Cause 2> L'ordinateur ne peut pas communiquer avec l'imprimante vers laquelle l'environnement de connexion de terminal le redirige.
Solution
Les paramètres du pare-feu, entre autres, peuvent être en cause.
Vérifiez les paramètres de communication du serveur, des clients, etc.
 Port incorrect
<Cause> L'imprimante est connectée à un port qu'elle ne prend pas en charge.
Solution
Vérifiez le port.
"Fonctionnement de base/Informations diverses pour Windows"
 Impossible d'imprimer
<Cause> L'impression est impossible en raison de restrictions de sécurité.
Solution
Contactez l'administrateur.
 Impossible d'imprimer
<Cause> Le délai de transfert des données vers l'imprimante s'est écoulé ou une erreur de transfert s'est produite.
Solution 1
Cliquez sur [ ] (Résoudre l'erreur) pour continuer à imprimer.
Toutefois, le message d'erreur peut réapparaître.
Solution 2
Annulez le travail d'impression, puis relancez l'impression.
"Annulation d'un travail"
 Vérifiez la sortie d'impression
<Cause> Vous risquez de ne pas pouvoir obtenir des résultats d'impression appropriés.
Solution 1
Cliquez sur [ ] (Résoudre l'erreur) pour continuer à imprimer.
Toutefois, le message d'erreur peut réapparaître.
Solution 2
Annulez le travail d'impression, puis relancez l'impression.
"Annulation d'un travail"
 Couvercle ouvert
<Cause> Le couvercle supérieur est ouvert.
Solution
Fermez le couvercle supérieur.
 Bourrage dans l'imprimante
<Cause> Un bourrage papier s'est produit.
Solution
Ne tirez pas trop fort sur le papier coincé, mais retirez-le en suivant la procédure décrite dans le manuel électronique.
"Bourrages papier"
 Espace disque insuffisant
<Cause> L'espace disque de l'ordinateur ou du serveur est insuffisant.
Solution
Supprimez les fichiers inutiles.
Cliquez sur [ ] (Résoudre l'erreur) pour continuer l'impression après suppression des fichiers superflus.
 Vérifiez l'imprimante
<Cause 1> La cartouche d'encre n'est pas installée.
Solution
Installez la cartouche d'encre.
"Remplacement de la cartouche d'encre"
<Cause 2> Il reste du papier dans l'imprimante.
Solution
Retirez le papier restant.
"Bourrages papier"
 Mémoire insuffisante
<Cause> La mémoire de l'ordinateur ou du serveur est insuffisante.
Solution
Fermez les applications inutiles.
 Il n'y a plus de papier ou il est impossible de charger le papier
<Cause> Le papier est épuisé ou n'a pas pu être alimenté correctement.
Solution
Chargez correctement le papier, puis appuyez sur la touche [  ] (Papier) de l'imprimante.
"Chargement du papier"
 Vérifiez le papier
<Cause> Le format papier du travail d'impression actuel est différent de celui du travail précédent.
Solution 1
Chargez le papier voulu, puis cliquez sur [ ] (Résoudre l'erreur).
"Chargement du papier"
Solution 2
Cliquez sur [ ] (Résoudre l'erreur) pour continuer l'impression sur le papier actuellement chargé.
