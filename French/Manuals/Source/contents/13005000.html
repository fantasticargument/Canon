L'impression est impossible
Les solutions à appliquer si l'impression est impossible varient selon votre environnement. Reportez-vous à la solution qui correspond à votre environnement.
Si vous utilisez l'imprimante avec une connexion directe
"L'impression est impossible (connexion USB)"
Si vous utilisez l'imprimante dans un environnement de serveur d'impression
Si votre ordinateur est le serveur d'impression
"L'impression est impossible (connexion USB)"
Si votre ordinateur est un ordinateur client
"L'impression ne fonctionne pas (via le serveur d'impression)"
