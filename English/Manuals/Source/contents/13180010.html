Paper Curls
Perform one of the following solutions according to the cause.
<Cause 1> Damp paper is used.
Solution 1
Replace the paper with new paper in an unopened package.
"Precautions for Paper"
Solution 2
Perform the following procedure in the printer driver when you are using plain paper (64 to 89 g/m²).
1.
Display the [Paper Source] sheet.
2.
Set [Paper Type] to [Plain Paper L].
Solution 3
Perform the following procedure in the printer driver.
1.
Display the [Finishing] sheet.
2.
Click [Advanced Settings].
3.
Set [Special Print Mode A] to [Special Settings B5].
Normally, use the printer with [Special Print Mode A] set to [Off]. Specify [Special Settings B5] only if the described problems occur.
Selecting [Special Settings B5] may slow down the print speed.
<Cause 2> Appropriate paper is not being used.
Solution
Replace the paper with that which can be used with this printer.
"Usable Paper"
