1. Масштабирование

  Некоторые приложения могут неправильно выполнять печать, если 
  выбрано масштабирование. В этом случае установите масштаб 
  равным 100%.
  Если в приложении имеется собственная функция масштабирования,
  установите масштаб в приложении.

================================================================

2. Разметка страницы

  В некоторых приложениях имеется функция печати нескольких 
  комплектов копий. Однако не следует использовать эту функцию, 
  если на вкладке [Параметры страницы] для параметра [Разметка 
  страницы] выбрана какая-либо настройка, кроме [1 на 1].
  Если эта функция используется в приложении, данные могут быть 
  скомпонованы неправильно.

================================================================

3. Страница-разделитель

  [Страница-разделитель] в Windows не может быть напечатана.

================================================================

4. Право пользователя на установку

  Чтобы установить драйвер принтера, войдите в систему в 
  качестве пользователя с правами администратора. Для установки 
  необходимо разрешение на Полный доступ.

================================================================

5. Загрузка и установка сетевого принтера 
   (Windows 2000/XP/Server 2003)

  Если сетевой принтер загружается и устанавливается в Windows 
  2000/XP/Server 2003, его может установить пользователь без 
  прав администратора.
  Если, однако, сетевой принтер устанавливается пользователем, 
  не имеющим прав администратора, то некоторые функции могут 
  быть недоступны вследствие ограничений системы защиты 
  операционной системы.

  [Недоступные функции]
  - Программа отмены установки не может быть зарегистрирована в 
    [Установка и удаление программ] ([Установка и удаление 
    программ] для Windows 2000).

================================================================

6. Установка многопользовательской среды 
   (Windows XP/Vista/7/Server 2008)

  Если этот принтер установлен в Windows XP/Vista/7/Server 2008,
  не следует устанавливать его для пользователя, подключенного с
  помощью функции "Быстрое переключение пользователей" в 
  Windows XP/Vista/7/Server 2008.
  При использовании функции "Быстрое переключение пользователей"
  перед установкой обязательно войдите в систему.

================================================================

7. Удаление сетевого принтера, который подключен к другому 
   пользователю

  Даже если запущена программа удаления, информация о сетевом 
  принтере, установленном на локальном компьютере другого 
  пользователя, не удаляется, а остается в операционной системе 
  в качестве частной информации для пользователя.
  Если другой пользователь установил принтер на локальный 
  компьютер через сеть, перед запуском программы отмены 
  установки убедитесь, что он удалил значок сетевого принтера из
  папки принтера.

================================================================

8. Ограничения сетевого принтера

  Если сетевой принтер установлен, то даже при изменении 
  назначения его подключения на локальный порт принтер не может 
  быть использован в качестве локального принтера.
  Если принтер используется в качестве локального, удалите 
  установленный сетевой принтер, выберите локальный порт, а 
  затем установите драйвер принтера заново.

================================================================

9. Перезагрузка Windows с оставшимся заданием печати

  Если необходимо перезагрузить Windows с оставшимся заданием 
  печати, приостановите задание печати в папке принтера. Потом 
  перезагрузите Windows.

================================================================

10. Защищенный режим Internet Explorer 
    (Windows Vista/7/Server 2008)

  Если в системе Windows Vista/7/Server 2008 запустить Internet 
  Explorer, щелкнув значок [Отображать руководство] при 
  установке с компакт-диска, то [Защищенный режим] будет в 
  состоянии [выкл.] независимо от настроек.
  Чтобы перевести [Защищенный режим] в состояние [вкл.], 
  закройте Internet Explorer и запустите его заново вручную.

================================================================

11. Символы для имени принтера (Windows Vista/7/Server 2008)

  Если в имени принтера используется символ "@", он становится 
  нераспознаваемым для операционной системы, поскольку его имя 
  некорректно, вследствие чего становятся невозможными печать, 
  работа и настройка.

================================================================

12. Регистрация пользовательских размеров бумаги 
    (Windows Vista/7/Server 2008)

  Чтобы изменить [Пользов. размер бумаги] на вкладке [Параметры 
  страницы] в драйвере принтера, войдите в систему в качестве 
  пользователя с правами администратора.

================================================================

13. Право на доступ к свойствам принтера 
    (Windows Vista/7/Server 2008)

  Если Вы выполняете вход в систему в качестве пользователя с 
  правами администратора, отличного от пользователя, 
  установившего драйвер принтера, Вы не сможете получить доступ 
  к свойствам принтера с правами администратора.
  Чтобы получить доступ к свойствам принтера, выполните одну из 
  указанных ниже процедур.

  - Windows Vista и Server 2008
    Щелкните правой кнопкой мыши значок принтера, а затем 
    выберите [Запуск от имени администратора].

  - Windows 7 и Server 2008 R2
    В свойствах принтера на вкладке [Общие] щелкните [Изменить 
    свойства].

================================================================

14. Печать тонких линий (Windows Vista/7/Server 2008)

  В некоторых приложениях толщине линии может быть присвоено 
  значение меньше 1 пт, однако фактическая толщина при печати 
  может быть эквивалентна толщине линии в 1 пт.

================================================================

15. Предостережения на случай, если используется Microsoft Excel

  При печати данных Microsoft Excel, содержащих объединенные 
  ячейки, символы данных могут не печататься.

  Поскольку вышеописанное явление возникает при изменении 
  параметров дисплея в диалоговых окнах [Настройки 
  (пользователи)] или [Настройки (администраторы)] в окне 
  состояния принтера следующим образом, будьте предусмотрительны
  при изменении параметров.

  - Установка для [Отобразить настройки времени] значения [При 
    запуске печати] (по умолчанию: [При возникновении ошибок])
  - Установка для [Развернуть окно при автоматическом 
    отображении] значения Вкл. (по умолчанию: Вкл.)

================================================================

16. Предостережения, касающиеся обновления Windows Vista до 
    Windows 7

  При обновлении операционной системы Windows Vista до Windows 7
  сначала удалите драйвер принтера, затем обновите операционную 
  систему, а после обновления — переустановите драйвер принтера.
  Если обновить операционную систему, предварительно не удалив 
  драйвер принтера, в его работе могут возникать сбои.
  В таком случае удалите драйвер принтера и переустановите его.

================================================================

17. Canon Driver Configuration Tool

  Этот драйвер поддерживает программу Canon Driver Configuration
  Tool версии 2.00 или следующих версий.
  Настройки драйвера принтера по умолчанию можно изменить для 
  создания пользовательских наборов драйверов с помощью Canon 
  Driver Configuration Tool.
  Для получения дополнительной информации об этой программе 
  обратитесь к руководству Canon Driver Configuration Tool 
  User's Guide.
