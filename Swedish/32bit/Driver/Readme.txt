================================================================

1. Skalning

  Vissa program kan inte skriva ut ordentligt om skalning 
  angetts. Om så är fallet ändrar du skalningen till 100 %.
  Om programmet innehåller funktioner för skalning ändrar du 
  skalningen i programmet.

================================================================

2. Sidlayout

  En del program innehåller en funktion för utskrift av flera 
  kopior. Du bör dock inte använda funktionen om en annan 
  inställning än [1 på 1] är vald för [Sidlayout] på fliken 
  [Sidformat].
  Om du använder funktionen i programmet kanske inte layouten på
  informationen blir rätt.

================================================================

3. Skiljesida

  [Skiljesida] i Windows kan inte skrivas ut.

================================================================

4. Användarbehörighet vid installationen

  Logga in som medlem med administratörsgruppen för att 
  installera skrivardrivrutinen. Fullständig behörighet krävs 
  för installationen.

================================================================

5. Ladda ner och installera en nätverksskrivare 
   (Windows 2000/XP/Server 2003)

  Om du laddar ner och installerar en nätverksskrivare på 
  Windows 2000/XP/Server 2003, kan du installera den som 
  användare utan administratörsbehörighet.
  Om nätverksskrivaren installeras av en användare utan 
  administratörsbehörighet kanske en del funktioner inte 
  fungerar på grund av den begränsade operativsystemssäkerheten.

  [Otillgängliga funktioner]
  - Avinstallationsprogrammet kan inte registreras i [Lägg till 
    eller ta bort program] ([Lägg till/ta bort program] för 
    Windows 2000).

================================================================

6. Installera i en fleranvändarmiljö 
   (Windows XP/Vista/7/Server 2008)

  Om skrivaren installeras i Windows XP/Vista/7/Server 2008 ska 
  du inte installera den för en användare som ändrats av 
  funktionen "Snabbt användarbyte" i Windows XP/Vista/7/Server 
  2008.
  När du använder funktionen "Snabbt användarbyte", se till att 
  logga in igen innan installationen.

================================================================

7. Avinstallera nätverksskrivaren som är ansluten till en annan 
   användare

  Även om du kör Avinstalleraren, tas inte informationen om 
  nätverksskrivaren som är installerad på en annan användares 
  lokala dator bort utan ligger kvar i operativsystemet som 
  särskild information för användaren.
  Om en annan användare har installerat skrivaren på den lokala 
  datorn via nätverket, se till att användaren tar bort ikonen 
  för nätverksskrivaren från skrivarmappen innan du kör 
  Avinstalleraren.

================================================================

8. Begränsning för nätverksskrivare

  Om en nätverksskrivare är installerad kan den inte användas 
  som en lokal skrivare, även om du ändrar anslutningsmålet till
  en lokal port.
  Om du använder skrivaren som en lokal skrivare, avinstallera 
  den installerade nätverksskrivaren, välj en lokal port och 
  installera sedan skrivardrivrutinen på nytt.

================================================================

9. Starta om Windows med ett återstående utskriftsjobb

  Om du behöver starta om Windows med ett återstående 
  utskriftsjobb, pausar du utskriftsjobbet från skrivarmappen. 
  Starta sedan om Windows.

================================================================

10. Skyddat läge för Internet Explorer
    (Windows Vista/7/Server 2008)

  Om du använder Windows Vista/7/Server 2008 och kör Internet 
  Explorer genom att klicka på ikonen för [Visa manual] etc. på 
  installationsskivan, ställs [Skyddat läge] på [Av] oberoende 
  av inställningarna.
  Om du vill ställa [Skyddat läge] på [På], avsluta Internet 
  Explorer en gång och kör det sedan manuellt.

================================================================

11. Tecken för skrivarnamnet (Windows Vista/7/Server 2008)

  Om tecknet "@" används i namnet för skrivaren, upptäcks inte 
  skrivaren av operativsystemet på grund av ett ogiltigt 
  skrivarnamn vilket gör att utskrift, drift och installation 
  inte fungerar.

================================================================

12. Registrera Anpassade pappersstorlekar 
    (Windows Vista/7/Server 2008)

  Se till att logga in som medlem i administratörsgruppen du 
  redigerar [Anpassad pappersstorlek] på fliken [Sidformat] i 
  skrivardrivrutinen.

================================================================

13. Åtkomstbehörighet till Skrivare - egenskaper 
    (Windows Vista/7/Server 2008)

  Om du loggar in som en medlem i administratörsgruppen som 
  skiljer sig från den som installerar skrivardrivrutinen, kan 
  du inte komma åt skrivare - egenskaper som medlem i 
  administratörsgruppen.
  Utför något av följande för att komma åt skrivaregenskaper.

  - Windows Vista och Server 2008
    Högerklicka på skrivarikonen och välj sedan [Kör som 
    administratör].

  - Windows 7 och Server 2008 R2
    Klicka på [Ändra egenskaper] på fliken [Allmänt] i 
    skrivaregenskaper.

================================================================

14. Skriva ut tunna linjer (Windows Vista/7/Server 2008)

  I vissa program kan tjockleken för en linje ställas in till 
  ett värde mindre än 1 pt men den verkliga tjockleken i 
  utskriften kan motsvara en linje med 1 pt.

================================================================

15. Föreskrifter för användning av Microsoft Excel

  När du skriver ut data i Microsoft Excel som innehåller 
  kombinerade celler, kan det hända att tecken i dessa data inte
  skrivs ut.

  Eftersom fenomenet ovan inträffar genom att du ändrar 
  visningsinställningarna i dialogrutan [Inställningar 
  (användare)] eller [Inställningar (administratörer)] i 
  fönstret Skrivarstatus enligt följande, bör du vara försiktig 
  när du ändrar inställningarna.

  - Ställa [Visa tidsinställningar] till [När utskrift startas] 
    (standard: [När fel inträffar])
  - Ställa [Maximera fönstret när det visas automatiskt] till På
    (standard: På)

================================================================

16. Försiktighetsåtgärder vid uppgradering av Windows Vista till
    Windows 7

  När du uppgraderar operativsystemet från Windows Vista till 
  Windows 7 avinstallerar du först skrivardrivrutinen, 
  uppgraderar operativsystemet och installerar sedan om 
  skrivardrivrutinen efter uppgraderingen.
  Om du uppgraderar operativsystemet utan att avinstallera 
  skrivardrivrutinen kan det hända att skrivardrivrutinen inte 
  fungerar ordentligt.
  Avinstallera i så fall skrivardrivrutinen en gång och 
  installera sedan om den igen.

================================================================
