Installera e-handboken
Installera e-handboken från den medföljande CD-ROM på datorn på följande sätt.
1
Sätt i den medföljande CD-ROM "User Software" i datorns CD-ROM-enhet.
2
Klicka på [Egen installation].
 Om skärmen ovan inte visas
 "Vanliga funktioner/Diverse information om Windows"
 Om dialogrutan [Spela upp automatiskt] öppnas
Klicka på [Kör AUTORUN.EXE].
 Om dialogrutan [Kontroll av användarkonto] öppnas
Klicka på [Ja] eller [Fortsätt].
3
Gör på följande sätt.
(1)
Avmarkera kryssrutan [Skrivardrivrutin].
(2)
Klicka på [Installera].
4
Läs licensavtalet och klicka sedan på [Ja].
→
Installationen startar.
Vänta tills följande skärmbild stängs. (Det kan ta lång tid att installera e-handboken.)
5
Klicka på [Nästa].
6
Klicka på [Avbryt].
När följande skärmbild öppnas kan du ta ur CD-ROM.
Öppna e-handboken
Dubbelklicka på [LBP6000 LBP6018 e-handbok] som har skapats på skrivbordet.
 När ett säkerhetsmeddelande visas

Gör på följande sätt.
1.
Klicka på informationsfältet och välj sedan [Tillåt innehåll som har blockerats] på snabbmenyn.
2.
Klicka på [Ja].
OBS!
Öppna e-handboken direkt från CD-ROM
Du kan även öppna e-handboken direkt från CD-ROM utan att installera den på datorn.
1.
Sätt i den medföljande CD-ROM "User Software" i datorns CD-ROM-enhet.
2.
Klicka på [Visa manualer].
3.
Klicka på knappen [] intill [e-handbok].
Ett säkerhetsmeddelande kan visas beroende på operativsystem.
Tillåt att innehållet visas.
