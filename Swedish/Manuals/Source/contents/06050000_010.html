Avbryta ett utskriftsjobb i utskriftskön
1
Visa utskriftskön.
När du visar hjälpen från skrivarstatusfönstret
1.
Klicka på [  ] i Windows aktivitetsfält och välj sedan skrivarens namn.
2.
Klicka på [  ] (Utskriftskö).
När du visar från mappen Skrivare
1.
Öppna mappen Skrivare.
"Vanliga funktioner/Diverse information om Windows"
2.
Dubbelklicka på den här skrivarens ikon.
2
Högerklicka på det utskriftsjobb du vill avbryta och välj [Avbryt] på snabbmenyn.
 Om ett utskriftsjobb inte visas
Andra användares utskriftsjobb visas inte. (I skrivarservermiljö visas alla utskriftsjobb på skrivarservern.)
OBS!
Mer information om utskriftskön
Se Windows Hjälp.
3
Klicka på [Ja] för att avbryta jobbet.
 Om du inte kan avbryta ett jobb
Du kan inte avbryta ett jobb vars sidor håller på att skrivas ut.
