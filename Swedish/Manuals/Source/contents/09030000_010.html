Konfigurera skrivarservern
Kontrollera inställningarna för nätverksmiljön innan du konfigurerar skrivarservern.
"Kontrollera inställningarna i nätverket"
1
Öppna mappen Skrivare.
"Vanliga funktioner/Diverse information om Windows"
2
Öppna fliken [Delning].
    
Högerklicka på ikonen för skrivaren och välj [Dela ut] på snabbmenyn.
 
1.
Högerklicka på ikonen för skrivaren och välj sedan [Egenskaper för skrivare] på snabbmenyn.
2.
Öppna fliken [Delning].
3
Gör på följande sätt.

Välj [Dela ut som]. Ändra [Resursnamn] om det behövs.
 
Välj [Dela ut den här skrivaren]. Ändra [Resursnamn] om det behövs.
  
Markera rutan [Dela ut den här skrivaren]. Ändra [Resursnamn] om det behövs.
Använd inte blanksteg eller specialtecken i resursnamnet.
 Om [Ändra delningsalternativ] visas
Klicka på [Ändra delningsalternativ].
Klicka på [Fortsätt] om dialogrutan [Kontroll av användarkonto] öppnas.
Installera sedan extradrivrutinen på följande sätt.
4
Sätt i den medföljande CD-ROM "User Software" i datorns CD-ROM-enhet och klicka sedan på [Ytterligare drivrutiner].
Om installationsskivan visas klickar du på [Avbryt].
OBS!
När en extra drivrutin uppdateras (ersättningsdrivrutin)
Gör på följande sätt.
1.
Avinstallera skrivardrivrutinen på skrivarservern.
"Avinstallera skrivardrivrutinen"
2.
Installera den nya skrivardrivrutinen på skrivarservern.
"Installera skrivardrivrutinen"
3.
Installera om extradrivrutinen.
5
Välj extradrivrutinen.
(1)
Välj extradrivrutinen.*
(2)
Klicka på [OK].
*
Välj extradrivrutinen enligt följande beroende på skrivarserverns operativsystem.
Skrivarserver
Välja extradrivrutin
32-bitars operativsystem
Markera kryssrutan [x64].
64-bitars operativsystem
(Windows XP och Server 2003)
Markera kryssrutan för [Windows 2000, Windows XP och Windows Server 2003] i kolumnen [Version].
64-bitars operativsystem
(Windows Vista, 7 och Server 2008)
Markera kryssrutan för [x86] i kolumnen [Processor].
 Om du inte är säker på om du använder ett 32- eller 64-bitars operativsystem (Windows Vista, 7 och Server 2008)
"Vanliga funktioner/Diverse information om Windows"
6
Klicka på [Bläddra].
7
Öppna mappen där skrivardrivrutinen finns.
Om ett 32-bitars operativsystem körs på skrivarservern
Öppna mapparna [Swedish] - [x64] - [Driver] på den medföljande CD-ROM.
Om ett 64-bitars operativsystem körs på skrivarservern
Öppna mapparna på den medföljande CD-ROM: [Swedish] - [32bit] - [Driver].
8
Öppna INF-filen.
(1)
Välj INF-filen.
(2)
Klicka på [Öppna].
9
Klicka på [OK] för att starta installationen av extradrivrutinen.
→
Vänta en liten stund tills fliken [Delning] visas automatiskt igen. (Det kan ta lång tid att installera extradrivrutinen.)
 Om dialogrutan [Kontroll av användarkonto] öppnas
Klicka på [Ja].
10
Klicka på [Stäng] eller [OK].
När följande skärmbild öppnas kan du ta ur CD-ROM.
→
[Tillstånd] och skrivarikonen ändras till dem för delningsinställningarna.
VIKTIGT!
Radera inställningarna för delad skrivare
Radera dem på följande sätt.

Välj [Ej delad] på fliken [Delning].
 
Välj [Dela inte ut den här skrivaren] på fliken [Delning].
  
Avmarkera kryssrutan [Dela ut den här skrivaren] på fliken [Delning].
Klicka på [Ändra delningsalternativ] om [Ändra delningsalternativ] visas.
Klicka på [Fortsätt] om dialogrutan [Kontroll av användarkonto] öppnas.
