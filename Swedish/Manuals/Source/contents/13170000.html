 Driftfel
<Orsak> Det är fel på skrivaren.
Lösning
Följ beskrivningen i "När <Driftfel> visas".
 Inkompatibel skrivare
<Orsak> Den skrivare som är ansluten kan inte användas.
Lösning
Anslut till en skrivare som kan användas.
 Kan inte kommunicera med servern
<Orsak> Klientdatorn kan inte kommunicera med skrivarservern.
Lösning
Kontrollera statusen för skrivarservern och datorn.
 Kommunikationsfel
<Orsak> USB-kabeln är inte ansluten eller skrivaren är inte påslagen.
Lösning 1
Anslut USB-kabeln.
Lösning 2
Slå på skrivaren.
 Fel i nätverkskort
<Orsak> Nätverkskabeln är inte ansluten till nätverksadaptern. Alternativt är inte nätverksadaptern och skrivaren PÅ.
Lösning 1
Anslut nätverkskabeln till nätverksadaptern.
Lösning 2
Sätt PÅ skrivaren och nätverksadaptern.
 Kan inte kommunicera med skrivaren
<Orsak 1> Eftersom funktionen för dubbelriktad kommunikation inte har aktiverats kan datorn inte kommunicera med skrivaren.
Lösning
Aktivera den dubbelriktade kommunikationen och starta om datorn och skrivaren.
"Vanliga funktioner/Diverse information om Windows"
<Orsak 2> Datorn kan inte kommunicera om skrivaren är omdirigerad i terminalmiljön.
Lösning
Brandväggsinställningar osv. kan vara orsaken.
Kontrollera kommunikationsinställningarna för servern, klienter osv.
 Felaktig port
<Orsak> Skrivaren har anslutits till en port som den inte stöder.
Lösning
Kontrollera porten.
"Vanliga funktioner/Diverse information om Windows"
 Det går inte att skriva ut
<Orsak> Utskrift kan inte göras på grund av säkerhetsbegränsningar.
Lösning
Kontakta din administratör.
 Det gick inte att skriva ut
<Orsak> Dataöverföringen till skrivaren överskred tidsgränsen eller så uppstod ett överföringsfel.
Lösning 1
Klicka på [ ] (Korrigera fel) för att fortsätta utskriften.
Felmeddelandet kan dock visas igen.
Lösning 2
Avbryt utskriftsjobbet och skriv sedan ut igen.
"Avbryta ett jobb"
 Kontrollera utskriften
<Orsak> Du kanske inte kan få ett godtagbart utskriftsresultat.
Lösning 1
Klicka på [ ] (Korrigera fel) för att fortsätta utskriften.
Felmeddelandet kan dock visas igen.
Lösning 2
Avbryt utskriftsjobbet och skriv sedan ut igen.
"Avbryta ett jobb"
 Den översta luckan är öppen
<Orsak> Översta luckan är öppen.
Lösning
Stäng den översta luckan.
 Pappersstopp inuti skrivaren
<Orsak> Ett papper har fastnat.
Lösning
Dra inte ut papperet som fastnat utan ta bort det genom att följa beskrivningen i e-handboken.
"Pappersstopp"
 Det finns inte tillräckligt med utrymme på hårddisken
<Orsak> Hårddiskutrymmet på datorn eller servern är otillräckligt.
Lösning
Ta bort onödiga filer.
Klicka på [ ] (Korrigera fel) för att återuppta utskriften efter att onödiga filer har tagits bort.
 Kontrollera skrivaren
<Orsak 1> Tonerkassetten har inte installerats.
Lösning
Sätt i tonerkassetten.
"Byte av tonerkassett"
<Orsak 2> Papper finns kvar inuti skrivaren.
Lösning
Ta bort resten av papperet.
"Pappersstopp"
 Otillräckligt minne
<Orsak> Minnet i datorn eller servern är otillräckligt.
Lösning
Avsluta program som inte behövs.
 Slut på papper eller papper kan inte matas
<Orsak> Papperet tog slut eller kunde inte matas ordentligt.
Lösning
Fyll på lämpligt papper och tryck sedan på [  ] knappen (Papper) på skrivaren.
"Fylla på papper"
 Kontrollera papperet
<Orsak> Pappersstorleken för aktuellt jobb skiljer sig från den för föregående jobb.
Lösning 1
Fyll på med lämpligt papper och klicka sedan på [ ] (Korrigera fel).
"Fylla på papper"
Lösning 2
Klicka på [ ] (Korrigera fel) för att skrivaren ska skriva ut på papperet som ligger i.
