Installera från mappen Skrivare
Installationen av skrivardrivrutinen från mappen Skrivare görs på olika sätt beroende på operativsystem.
Följ nedanstående anvisningar beroende på operativsystem.
Windows Vista, 7 och Server 2008
Windows XP och Server 2003
Windows 2000
Windows Vista, 7 och Server 2008
1
Starta datorn och logga sedan in till Windows.
2
Öppna mappen Skrivare.
"Vanliga funktioner/Diverse information om Windows"
3
Klicka på [Lägg till en skrivare].
4
Klicka på [Lägg till en nätverksskrivare, trådlös skrivare eller Bluetooth-skrivare].
5
Klicka på [Skrivaren jag vill använda finns inte med i listan].
6
Gör på följande sätt.
(1)
Välj [Välj en delad skrivare efter namn].
(2)
Klicka på [Nästa].
7
Välj en delad skrivare.
(1)
Välj skrivaren som delas via skrivarservern.
(2)
Klicka på [Välj].
OBS!
Ange nätverkssökvägen direkt
Ange nätverkssökvägen som "\\skrivarserverns namn (datornamnet för skrivarservern)\namnet på skrivaren".
8
Klicka på [Installera drivrutin].
 Om dialogrutan [Kontroll av användarkonto] öppnas
Klicka på [Ja] eller [Fortsätt].
9
Klicka på [Nästa].
När [Använd som standardskrivare] visas väljer du om skrivaren ska användas som standarskrivare eller inte. (Windows Vista och Server 2008)
10
Klicka på [Slutför].
Klicka på [Skriv ut en testsida] om du vill skriva ut en testsida.
När [Använd som standardskrivare] visas väljer du om skrivaren ska användas som standarskrivare eller inte. (Windows 7 och Server 2008 R2)
Windows XP och Server 2003
1
Starta datorn och logga sedan in till Windows.
2
Öppna mappen Skrivare.
"Vanliga funktioner/Diverse information om Windows"
3
Öppna [Guiden Lägg till skrivare].

Klicka på [Lägg till en skrivare].

Dubbelklicka på [Lägg till skrivare].
4
Klicka på [Nästa].
5
Välj en skrivartyp.
(1)
Välj [En nätverksskrivare eller skrivare som är ansluten till en annan dator].
(2)
Klicka på [Nästa].
6
Gör på följande sätt.
(1)
Markera [Anslut till den här skrivaren].
(2)
Klicka på [Nästa].
7
Välj en delad skrivare.
(1)
Välj skrivaren som delas via skrivarservern.
(2)
Klicka på [Nästa].
 Om dialogrutan [Anslut till skrivare] öppnas
Följ anvisningarna i meddelandet.
OBS!
Ange nätverkssökvägen direkt
Ange nätverkssökvägen som "\\skrivarserverns namn (datornamnet för skrivarservern)\namnet på skrivaren".
8
När följande skärmbild visas väljer du om du vill använda den här skrivaren som standardskrivare eller inte.
(1)
Om du vill använda skrivaren som standardskrivare väljer du [Ja].
(2)
Klicka på [Nästa].
9
Klicka på [Slutför].

Klicka på [Nästa].
7

(1)
Välj skrivaren som delas via skrivarservern.
(2)
Klicka på [Nästa].

Välj om skrivaren ska användas som standardskrivare eller inte.
(1)
Om du vill använda skrivaren som standardskrivare väljer du [Ja].
(2)
Klicka på [Nästa].
9
Klicka på [Slutför].
