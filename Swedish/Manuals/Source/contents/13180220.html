Utskriften ser inte ut som på skärmbilden
Gör på följande sätt.
<Orsak> Det här problemet kan inträffa i vissa program beroende på hur utskriftsdata bearbetas.
Lösning
Gör på följande sätt i skrivardrivrutinen för att eventuellt lösa problemet.
1.
Öppna fliken [Efterbehandling].
2.
Klicka på [Avancerade inställningar].
3.
Ändra inställningen för [EMF-buffring].
Om [EMF-buffring] står på [På], kan det hända att skrivarstatusfönstret inte visas automatiskt. Visa det i så fall manuellt.
(Mer information om hur du visar skrivarstatusfönstret finns i "Skrivarstatusfönstret".)
