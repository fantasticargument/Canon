Avinstallera USB-klassdrivrutinen
Avinstallera USB-klassdrivrutinen under följande omständigheter.
Om skrivardrivrutinen har installerats via en USB-anslutning och inte kan avinstalleras helt
Om du inte kan avinstallera skrivardrivrutinen ordentligt trots att du har installerat om den
Var noga med att göra på följande sätt när du avinstallerar USB-klassdrivrutinen.
Om du tar bort USB-klassdrivrutinen installeras den automatiskt igen via USB-anslutningen.
1
Kontrollera följande.
Skrivaren och datorn är anslutna med en USB-kabel.
Skrivaren har startats.
2
Gör på följande sätt.

Välj [Inställningar] → [Kontrollpanelen] från [Start]-menyn och dubbelklicka sedan på [Lägg till/ta bort program].

Välj [Kontrollpanelen] på [Start]-menyn och klicka sedan på [Lägg till eller ta bort program].
  
Välj [Kontrollpanelen] på [Start]-menyn och klicka sedan på [Avinstallera ett program].

Välj [Kontrollpanelen] → [Lägg till eller ta bort program] på [Start]-menyn.

Välj [Kontrollpanelen] på [Start]-menyn och dubbelklicka sedan på [Program och funktioner].
3
Kontrollera att [Canon LBP6000/LBP6018] inte visas.
(1)
Kontrollera att [Canon LBP6000/LBP6018] inte visas i dialogrutan.
(2)
Klicka på [  ] för att stänga dialogrutan.
 Om [Canon LBP6000/LBP6018] inte visas
Ta bort [Canon LBP6000/LBP6018] i dialogrutan. (Avinstallera skrivardrivrutinen.)
"Starta avinstallationen från [Kontrollpanelen]"
"Avinstallera skrivardrivrutinen"
4
Öppna [Enhetshanteraren].

1.
Välj [Inställningar] → [Kontrollpanelen] på [Start]-menyn.
2.
Dubbelklicka på [System]-ikonen.
3.
Klicka på fliken [Maskinvara] → [Enhetshanteraren].

1.
Välj [Kontrollpanelen] på [Start]-menyn.
2.
Klicka på [Prestanda och underhåll] → [System].
3.
Klicka på fliken [Maskinvara] → [Enhetshanteraren].
  
1.
Välj [Kontrollpanelen] på [Start]-menyn.
2.
Klicka på [Maskinvara och ljud] eller [Maskinvara] → [Enhetshanteraren].

1.
Välj [Kontrollpanelen] → [System] på [Start]-menyn.
2.
Klicka på fliken [Maskinvara] → [Enhetshanteraren].

1.
Välj [Kontrollpanelen] på [Start]-menyn.
2.
Dubbelklicka på [Enhetshanteraren].
 Om dialogrutan [Kontroll av användarkonto] öppnas
Klicka på [Ja] eller [Fortsätt].
5
Dubbelklicka på [USB-styrenheter].
6
Högerklicka på [USB Printing Support] och välj sedan [Avinstallera] på snabbmenyn.
Installera inte drivrutiner för några andra enheter. Om du råkar ta bort dem av misstag är det inte säkert att Windows fungerar ordentligt.
Om USB-klassdrivrutinen inte är rätt installerad visas inte [USB Printing Support]. Klicka sedan på [ ] för att stänga dialogrutan [Enhetshanteraren].
 Om USB-klassdrivrutinen finns under [Andra enheter]
Välj USB-klassdrivrutinen som visas under [Okänd enhet] och avinstallera den sedan.
7
Klicka på [OK].
8
Klicka på [ ] för att stänga dialogrutan [Enhetshanteraren].
9
Koppla loss USB-kabeln från datorn och starta sedan om Windows.
