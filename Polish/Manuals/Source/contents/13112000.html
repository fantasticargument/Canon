Odinstalowanie sterownika klasy USB
Odinstaluj sterownik klasy USB w następujących okolicznościach.
Jeśli nie można poprawnie odinstalować zainstalowanego sterownika drukarki za pomocą łącza USB
Jeśli nie można prawidłowo zainstalować sterownika drukarki nawet po jego ponownym zainstalowaniu
Podczas dezinstalacji sterownika klasy USB wykonaj następującą procedurę.
Jeżeli usuniesz sterownik klasy USB, zostanie on zainstalowany automatycznie przy połączeniu USB.
1
Sprawdź następujące elementy.
Drukarka i komputer połączone są kablem USB.
Drukarka została włączona.
2
Wykonaj następującą procedurę.

Z menu [Start] wybierz [Ustawienia] → [Panel sterowania] i kliknij dwukrotnie [Dodaj/Usuń programy].

Z menu [Start] wybierz [Panel sterowania], a następnie kliknij [Dodawanie lub usuwanie programów].
  
Z menu [Start] wybierz [Panel sterowania], a następnie kliknij [Odinstaluj program].

Z menu [Start] wybierz [Panel sterowania] → [Dodawanie lub usuwanie programów].

Z menu [Start] wybierz [Panel sterowania] i kliknij dwukrotnie [Programy i funkcje].
3
Upewnij się, że pozycja [Canon LBP6000/LBP6018] nie jest wyświetlana.
(1)
Upewnij się, że pozycja [Canon LBP6000/LBP6018] nie jest wyświetlana w oknie dialogowym.
(2)
Kliknij [  ], aby zamknąć okno dialogowe.
 Jeśli pozycja [Canon LBP6000/LBP6018] jest wyświetlana
Usuń pozycję [Canon LBP6000/LBP6018] w oknie dialogowym. (Odinstaluj sterownik drukarki.)
"Uruchamianie dezinstalatora z okna [Panel sterowania]"
"Odinstalowanie sterownika drukarki"
4
Wyświetl [Menedżer urządzeń].

1.
Z menu [Start] wybierz [Ustawienia] → [Panel sterowania].
2.
Kliknij dwukrotnie ikonę [System].
3.
Kliknij zakładkę [Sprzęt] → [Menedżer urządzeń].

1.
Z menu [Start] wybierz [Panel sterowania].
2.
Kliknij [Wydajność i konserwacja] → [System].
3.
Kliknij zakładkę [Sprzęt] → [Menedżer urządzeń].
  
1.
Z menu [Start] wybierz [Panel sterowania].
2.
Kliknij [Sprzęt i dźwięk] lub [Sprzęt] → [Menedżer urządzeń].

1.
Z menu [Start] wybierz [Panel sterowania] → [System].
2.
Kliknij zakładkę [Sprzęt] → [Menedżer urządzeń].

1.
Z menu [Start] wybierz [Panel sterowania].
2.
Kliknij dwukrotnie [Menedżer urządzeń].
 Jeśli wyświetlone zostanie okno dialogowe [Kontrola konta użytkownika]
Kliknij [Tak] lub [Kontynuuj].
5
Kliknij dwukrotnie [Kontrolery uniwersalnej magistrali szeregowej].
6
Prawym przyciskiem myszy kliknij [Uniwersalna drukarka USB] i z menu podręcznego wybierz [Odinstaluj].
Nie instaluj żadnych sterowników innych urządzeń. Jeśli przypadkowo odinstalujesz któryś ze sterowników, Windows może przestać działać prawidłowo.
Jeżeli sterownik klasy USB nie jest zainstalowany prawidłowo, pozycja [Uniwersalna drukarka USB] nie jest wyświetlana. Następnie kliknij [ ], aby zamknąć okno dialogowe [Menedżer urządzeń].
 Jeśli sterownik klasy USB znajduje się w opcji [Inne urządzenia]
Wybierz sterownik klasy USB wyświetlony jako [Nieznane urządzenie] i zainstaluj go.
7
Kliknij [OK].
8
Kliknij [  ], aby zamknąć okno dialogowe [Menedżer urządzeń].
9
Odłącz kabel USB od komputera i uruchom ponownie system Windows.
