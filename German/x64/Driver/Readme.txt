1. Skalieren

  Einige Anwendungen drucken nicht korrekt, wenn eine Skalierung
  angegeben ist. Stellen Sie in diesem Fall die Skalierung auf 
  100 % ein.
  Verfügt die Anwendung über eine interne Skalierungsfunktion, 
  sollten Sie diese verwenden.

================================================================

2. Seitenlayout

  Einige Anwendungen bieten eine Funktion zum Drucken mehrerer 
  Exemplare. Verwenden Sie jedoch diese Funktion nicht, wenn 
  eine andere Einstellung als [1 auf 1] für [Seitenlayout] auf 
  der Registerkarte [Seite einrichten] ausgewählt wurde.
  Wenn die Funktion in der Anwendung gewählt wird, werden die 
  Daten möglicherweise nicht ordnungsgemäß ausgegeben.

================================================================

3. Trennseite

  Die [Trennseite] kann unter Windows nicht gedruckt werden.

================================================================

4. Benutzerberechtigung zur Installation

  Für die Installation des Druckertreibers melden Sie sich als 
  Benutzer mit Administratorberechtigungen an. Für die 
  Installation ist eine Vollzugriffsberechtigung erforderlich.

================================================================

5. Herunterladen und Installieren eines Netzwerkdruckers 
   (Windows 2000/XP/Server 2003)

  Wenn Sie einen Netzwerkdrucker unter Windows 2000/XP/Server 
  2003 herunter laden und installieren, können Sie den Drucker 
  als Benutzer ohne Administratorberechtigungen installieren.
  Wenn der Netzwerkdrucker von einem Benutzer ohne 
  Administratorberechtigungen installiert wird, sind jedoch 
  möglicherweise einige Funktionen aufgrund der 
  Sicherheitsbeschränkungen des Betriebssystems nicht verfügbar.

  [Nicht verfügbare Funktionen]
  - Das Deinstallationsprogramm kann in [Software] nicht 
    registriert werden ([Software] für Windows 2000).

================================================================

6. Installation in einer Mehrbenutzerumgebung 
   (Windows XP/Vista/7/Server 2008)

  Wenn dieser Drucker unter Windows XP/Vista/7/Server 2008 
  installiert wird, sollte er nicht für den mit der Windows 
  XP/Vista/7/Server 2008-Funktion für "Schnelle 
  Benutzerumschaltung" aktivierten Benutzer installiert werden.
  Bei Verwendung der Funktion für "Schnelle Benutzerumschaltung"
  müssen Sie sich vor der Installation neu anmelden.

================================================================

7. Deinstallieren des Netzwerkdruckers, der mit einem anderen 
   Benutzer verbunden ist

  Auch wenn Sie das Deinstallationsprogramm ausführen, werden 
  die Informationen zum Netzwerkdrucker, der auf dem lokalen 
  Computer eines anderen Benutzers eingerichtet ist, nicht 
  gelöscht, sondern bleiben als besondere Informationen für den 
  Benutzer auf dem Betriebssystem erhalten.
  Wenn ein anderer Benutzer den Drucker auf seinem lokalen 
  Computer über das Netzwerk eingerichtet hat, stellen Sie 
  sicher, dass der Benutzer das Netzwerkdruckersymbol aus dem 
  Druckerordner löscht, bevor Sie das Deinstallationsprogramm 
  ausführen.

================================================================

8. Beschränkungen des Netzwerkdruckers

  Ein eingerichteter Netzwerkdrucker kann auch dann nicht als 
  lokaler Drucker verwendet werden, wenn Sie einen lokalen 
  Anschluss als Ziel angeben.
  Wenn Sie den Drucker als lokalen Drucker verwenden, 
  deinstallieren Sie den installierten Netzwerkdrucker, wählen 
  Sie einen lokalen Anschluss aus, und installieren Sie dann den
  Druckertreiber erneut.

================================================================

9. Windows-Neustart bei aktivem Druckauftrag

  Wenn Sie Windows neu starten müssen, aber noch ein 
  Druckauftrag vorhanden ist, halten Sie den Druckauftrag im 
  Druckerordner an. Starten Sie Windows anschließend neu.

================================================================

10. Der geschützte Modus des Internet Explorer 
    (Windows Vista/7/Server 2008)

  Wenn Sie Windows Vista/7/Server 2008 verwenden und den 
  Internet Explorer ausführen, indem Sie in der CD-ROM-
  Installation auf das Symbol für [Handbuch anzeigen] etc. 
  klicken, wird [Geschützter Modus] unabhängig von den 
  Einstellungen auf [Inaktiv] gesetzt.
  Wenn Sie [Geschützter Modus] auf [Aktiv] setzen möchten, 
  beenden Sie den Internet Explorer, und führen Sie ihn 
  anschließend manuell erneut aus.

================================================================

11. Zeichen für den Druckernamen (Windows Vista/7/Server 2008)

  Wenn im Namen des Druckers das Zeichen "@" verwendet wird, 
  kann der Drucker aufgrund eines ungültigen Druckernamens vom 
  Betriebssystem nicht mehr erkannt werden, was dazu führt, dass
  Druckvorgänge, Operationen und Konfigurationen nicht möglich 
  sind.

================================================================

12. Registrieren von benutzerdefinierten Papierformaten 
    (Windows Vista/7/Server 2008)

  Melden Sie sich unbedingt als Benutzer mit 
  Administratorberechtigungen an, wenn Sie [Benutzerdefiniertes 
  Papierformat] auf der Registerkarte [Seite einrichten] im 
  Druckertreiber bearbeiten möchten.

================================================================

13. Zugriffsrechte für die Druckereigenschaften 
    (Windows Vista/7/Server 2008)

  Wenn Sie sich als Benutzer mit Administratorberechtigungen 
  anmelden, der nicht mit dem identisch ist, der den 
  Druckertreiber installiert hat, können Sie auf die 
  Druckereigenschaften nicht als Benutzer mit 
  Administratorberechtigungen zugreifen.
  Führen Sie eine der folgenden Maßnahmen durch, um auf die 
  Druckereigenschaften zuzugreifen.

  - Windows Vista und Server 2008
    Klicken Sie mit der rechten Maustaste auf das Druckersymbol,
    und wählen Sie dann [Als Administrator ausführen] aus.

  - Windows 7 und Server 2008 R2
    Klicken Sie in den Druckereigenschaften auf der 
    Registerkarte [Allgemein] auf [Eigenschaften ändern].

================================================================

14. Drucken von dünnen Linien (Windows Vista/7/Server 2008)

  In einigen Anwendungen kann die Dicke einer Linie auf einen 
  Wert kleiner als 1 Punkt gesetzt werden, die tatsächliche 
  Dicke auf dem Ausdruck entspricht jedoch möglicherweise der 
  einer Linie mit dem Wert 1 Punkt.

================================================================

15. Vorsichtshinweise für die Verwendung von Microsoft Excel

  Wenn Sie Microsoft Excel-Daten drucken, die kombinierte Zellen
  enthalten, werden Zeichen in den Daten möglicherweise nicht 
  gedruckt.

  Da das oben beschriebene Phänomen auftritt, wenn Sie die 
  Anzeigeeinstellungen im Statusfenster des Druckers im 
  Dialogfeld [Voreinstellungen (Benutzer)] oder 
  [Voreinstellungen (Administratoren)] folgendermaßen ändern, 
  sollten Sie bei der Änderung der Einstellungen vorsichtig 
  sein.

  - Wenn Sie [Zeiteinstellungen anzeigen] auf [Beim Starten des 
    Druckvorgangs] setzen (Standardwert: [Beim Auftreten von 
    Fehlern])
  - Wenn Sie [Fenster bei automatischer Anzeige als Vollbild 
    anzeigen] auf Ein setzen (Standardwert: Ein)

================================================================

16. Vorsichtsmaßnahmen für die Durchführung eines Upgrades von 
    Windows Vista auf Windows 7

  Wenn Sie für das Betriebssystem ein Upgrade von Windows Vista 
  auf Windows 7 durchführen, deinstallieren Sie zunächst den 
  Druckertreiber, führen Sie dann für das Betriebssystem das 
  Upgrade durch, und installieren Sie nach der Durchführung des 
  Upgrades den Druckertreiber erneut.
  Wenn Sie für das Betriebssystem ein Upgrade durchführen, ohne 
  den Druckertreiber zu deinstallieren, wird der Druckertreiber 
  möglicherweise nicht ordnungsgemäß funktionieren.
  Deinstallieren Sie in diesem Fall den Druckertreiber einmal, 
  und installieren Sie ihn anschließend erneut.
