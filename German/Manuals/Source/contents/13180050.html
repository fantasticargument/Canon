Es sind weiße Flecken zu sehen
Führen Sie eine der folgenden Maßnahmen entsprechend der Ursache durch.
<Ursache 1> Es wird kein geeignetes Papier verwendet.
Abhilfe
Ersetzen Sie das Papier durch Papier, das mit diesem Drucker verwendet werden kann.
"Verwendbares Papier"
<Ursache 2> Es wird feuchtes Papier verwendet.
Abhilfe
Ersetzen Sie das Papier durch neues Papier aus einer ungeöffneten Verpackung.
"Vorsichtsmaßnahmen bei Papier"
<Ursache 3> Die Trommel in der Tonerpatrone ist verschlissen.
Abhilfe
Ersetzen Sie die Tonerpatrone durch eine neue.
"Austausch der Tonerpatrone"
<Ursache 4> Wenn Sie ein Dokument mit hoch kontrastierender Dichte drucken, können an den Konturen von Text oder Bildern weiße Flecken zu sehen sein.
Abhilfe
Gehen Sie im Druckertreiber folgendermaßen vor.
1.
Zeigen Sie die Registerkarte [Endverarbeitung] an.
2.
Klicken Sie auf [Erweiterte Einstellungen].
3.
Setzen Sie [Spezieller Druckmodus B] auf [Spezialeinstellungen 2].
Wenn Sie [Spezialeinstellungen 2] angeben, verringert sich die Druckdichte im Vergleich zur Angabe von [Aus].
<Ursache 5> Abhängig von der verwendeten Anwendung oder den verwendeten Daten können gedruckte Seiten weiße Flecken aufweisen.
Abhilfe
Gehen Sie im Druckertreiber folgendermaßen vor.
1.
Zeigen Sie die Registerkarte [Endverarbeitung] an.
2.
Klicken Sie auf [Erweiterte Einstellungen].
3.
Setzen Sie [Rendering-Modus A] auf [Modus 2].
Abhängig von der verwendeten Anwendung oder den verwendeten Daten kann sich durch Angabe von [Modus 2] die Druckerleistung verringern.
