No se reconoce automáticamente la impresora (Instalación simple)
Ejecute una de las siguientes soluciones de acuerdo con la causa.
<Causa 1> La impresora no está encendida.
Solución
Encienda la impresora.
<Causa 2> El cable USB no está conectado adecuadamente.
Solución
Asegúrese de que la impresora y el ordenador están conectados adecuadamente con el cable USB.
"Conexión de la impresora y el ordenador"
<Causa 3> El cable USB ya estaba conectado y la impresora estaba encendida antes de instalar el controlador de impresora.
Solución
Lleve a cabo el siguiente procedimiento.
1.
Apague la impresora.
2.
Desconecte el cable USB.
3.
Conecte de nuevo el cable USB.
4.
Encienda la impresora mientras aparece la siguiente pantalla.
<Causa 4> Se está utilizando un cable USB inapropiado.
Solución
Utilice un cable USB que sea adecuado para el entorno de interfaz USB de la impresora y que tenga el siguiente símbolo.
A continuación se describe el entorno de interfaz USB de esta impresora.
USB de alta velocidad
USB
<Causa 5> El controlador de clase USB está instalado.
Solución
Desinstale el controlador de clase USB.
"Desinstalación del controlador clase USB"
