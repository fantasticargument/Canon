Instalación con Plug and Play
1
Conecte el cable USB.
"Conexión de la impresora y el ordenador"
2
Encienda la impresora.
3
Encienda el ordenador y, a continuación, inicie una sesión en Windows como usuario con derechos administrativos.
→
Aparecerá la pantalla de instalación automática de Plug and Play.
 Si el controlador de impresora se instala automáticamente
Si instaló anteriormente el controlador de esta impresora, el controlador se instalará automáticamente.
Si desea actualizar el controlador de impresora (o instalarlo manualmente), instálelo después de haberlo desinstalado.
 "Desinstalación del controlador de impresora"
 Si aparece la siguiente pantalla

Tal vez no pueda instalar adecuadamente el controlador de impresora debido a la restricción de Windows ni siquiera después de detectar la impresora con Plug and Play.
Haga clic en [Cerrar] y, a continuación, vuelva a instalar el controlador de impresora en "Instalación simple".
4
Siga las instrucciones en pantalla para instalar el controlador de impresora.
NOTA
Cuando seleccione el archivo o la carpeta en los cuales está incluido el controlador de impresora.
Seleccione uno de los siguientes archivos y carpetas.
Seleccione [CNABCSTK.INF] en las carpetas [Spanish] - [32bit] - [Driver] del CD-ROM suministrado.

   
Para un sistema operativo de 32 bits
Seleccione las carpetas del CD-ROM suministrado: [Spanish] - [32bit] - [Driver].

Para un sistema operativo de 64 bits
Seleccione las carpetas del CD-ROM suministrado: [Spanish] - [x64] - [Driver].
Si no está seguro si está utilizando Windows Vista/Server 2008 de 32 o de 64 bits, consulte "Operaciones básicas/Diversas informaciones para Windows".
Si aparece la pantalla para seleccionar el controlador de impresora
Aunque se seleccione cualquier controlador de impresora, no habrá problemas con la instalación ya que se instalará el mismo controlador.
Una vez finalizada la instalación del controlador de impresora, se creará un icono y una carpeta para esta impresora.
"Después de la instalación"
