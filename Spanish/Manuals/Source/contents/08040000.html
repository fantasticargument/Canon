Impresión de formato (impresión de N páginas)
Es posible imprimir varias páginas en una sola hoja.
Éste es un método de impresión al que se le conoce con el nombre de impresión intensiva o Impresión de N páginas.
IMPORTANTE
Acerca de la función de aplicaciones para imprimir un documento como si fuese un juego intercalado de copias
Algunas aplicaciones proporcionan una función para imprimir un documento como si fuese un juego intercalado de copias. No utilice esta función cuando imprima varias páginas en una sola hoja. El documento podría no imprimirse correctamente.
1
Abra la pantalla de configuración del controlador de impresora.
"Apertura de la pantalla de configuración"
2
Especifique el valor para [Formato de página].
(1)
Abra la ficha [Configuración de página].
(2)
Seleccione el número de páginas que desea imprimir en una sola hoja.
 Si no puede especificar el valor
"No puede cambiarse la configuración del controlador de impresora"
NOTA
Utilización de la Vista preliminar para especificar la configuración
También podrá especificar un formato de página haciendo repetidamente clic en la imagen del documento o haciendo clic con el botón derecho del ratón en la Vista preliminar cuando esté seleccionado [ ].
3
Especifique el formato del orden de páginas en el papel.
4
Haga clic en [Aceptar].
Para obtener una información detallada sobre el procedimiento básico de impresión, consulte "Procedimiento básico de impresión".
