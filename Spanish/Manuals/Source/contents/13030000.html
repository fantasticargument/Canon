No se puede realizar impresión (a través del servidor de impresión)
Si no puede imprimir cuando la impresora se utiliza a través del servidor de impresión, compruebe el problema siguiendo las preguntas que se presentan a continuación.
<Pregunta 1> ¿La impresora está encendida? (¿está iluminado el indicador (Alimentación)?)
ENCENDIDA
A la <Pregunta 2>
APAGADA
Encienda la impresora.
No se enciende
"La impresora no se enciende"
<Pregunta 2> ¿Está parpadeando el indicador  (Papel)?
Parpadeando
Cargue el papel correctamente y, a continuación, pulse la tecla [  ] (Papel).
"Carga de papel"
No parpadeando
A la <Pregunta 3>
<Pregunta 3> ¿Está correctamente conectada la impresora al servidor de impresión?
Conectada
A la <Pregunta 4>
No conectada
Conecte el cable del modo apropiado.
"Conexión de la impresora y el ordenador"
<Pregunta 4> ¿Puede conectarse el ordenador al servidor de impresión o a la impresora compartida?
Puede conectarse
A la <Pregunta 5>
No puede conectarse
"No se puede conectar al servidor de impresión"
"No se puede acceder a la impresora compartida"
<Pregunta 5> Verifique los puntos siguientes.
El servidor de impresión está conectado a la red correctamente.
El controlador adicional (controlador alternativo) está actualizado adecuadamente.
"Configuración del servidor de impresión"
La impresión se puede realizar desde el servidor de impresión.

Si la impresión no funciona, proceda con la <Pregunta 6>.
<Pregunta 6> ¿Puede imprimir una página de prueba en Windows?
"Operaciones básicas/Diversas informaciones para Windows"
Se puede imprimir
No hay ningún problema con la impresora ni con el controlador.
Compruebe las preferencias de impresión en la aplicación.
No se puede imprimir
A la <Pregunta 7>
<Pregunta 7> ¿Se muestra la impresora del servidor de impresión en la red?
"Operaciones básicas/Diversas informaciones para Windows"
Se muestra
A la <Pregunta 8>
No se muestra
Pregunte a su administrador de red sobre el problema.
<Pregunta 8> Instale el controlador de impresora de nuevo.
"Desinstalación del controlador de impresora"
"Instalación en los clientes"
Cuando instale de nuevo el controlador de impresora, cambie el método de instalación.
Si ejecutó la instalación local (instalación del controlador de impresora con el CD-ROM)
→ Cambie el método de instalación a la instalación mediante descarga (instalación del controlador de impresora sin el CD-ROM).
Si ejecutó la instalación mediante descarga (instalación del controlador de impresora sin el CD-ROM)
→ Cambie el método de instalación a la instalación local (instalación del controlador de impresora con el CD-ROM).

Si la impresión no funciona, proceda con la <Pregunta 9>.
<Pregunta 9> ¿Se puede imprimir desde otro cliente?
Se puede imprimir
Desactive el software residente, incluido el software de seguridad.
No se puede imprimir
Póngase en contacto con su distribuidor autorizado local de Canon.
