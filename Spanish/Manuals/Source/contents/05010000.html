Instalación del controlador de impresora
El controlador de impresora es un software necesario para imprimir con esta impresora.
Para obtener una información detallada sobre el entorno del sistema requerido para utilizar el controlador de impresora, consulte "Requisitos del sistema para imprimir".
El procedimiento para instalar el controlador de impresora varía dependiendo del entorno del sistema. Consulte lo siguiente de acuerdo con su entorno del sistema.
Si conecta la impresora y el ordenador con un cable USB
Existen los dos modos siguientes para instalar el controlador de impresora.
<Instalación simple>
Instala el controlador de impresora desde el CD-ROM suministrado (Instalación de CD-ROM). El e-Manual también se instala simultáneamente.
"Instalación simple"
<Plug and Play>
Detecta la impresora con Plug and Play e instala el controlador de impresora seleccionándolo manualmente desde el CD-ROM suministrado.
"Instalación con Plug and Play"
Si utiliza la impresora en el entorno de servidor de impresión
Si su ordenador es el servidor de impresión
Para obtener una información detallada sobre el procedimiento para instalar el controlador de impresora, consulte la sección "Si conecta la impresora y el ordenador con un cable USB Cable" anterior.
Una vez finalizada la instalación, configure el servidor de impresión.
"Configuración del servidor de impresión"
Si su ordenador es un ordenador cliente
Existen los dos modos siguientes para instalar el controlador de impresora.
<Instalación local>
Podrá instalar el controlador de impresora utilizando el CD-ROM suministrado.
"Instalación simple"
<Instalación mediante descarga>
Podrá instalar el controlador de impresora sin utilizar el CD-ROM suministrado sino descargándolo del servidor de impresión. Existen los dos modos siguientes para descargar e instalar el controlador de impresora.
"Instalación desde la carpeta de impresoras"
"Instalación desde el [Explorador de Windows]"
