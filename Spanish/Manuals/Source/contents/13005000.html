No se puede realizar impresión
Las soluciones para cuando no se puede realizar impresión varían en función del entorno. Consulte las soluciones de acuerdo con su entorno.
Si está utilizando la impresora con conexión directa
"No se puede realizar impresión (conexión USB)"
Si está utilizando la impresora en el entorno de servidor de impresión
Si su ordenador es el servidor de impresión
"No se puede realizar impresión (conexión USB)"
Si su ordenador es un ordenador cliente
"No se puede realizar impresión (a través del servidor de impresión)"
