1. Escala

  Algunas aplicaciones podrían no imprimir correctamente si se 
  especifica una escala. En este caso, establezca la escala en 
  100%.
  Si la aplicación posee su propia función de escala, determine 
  la escala desde la aplicación.

================================================================

2. Formato de página

  Algunas aplicaciones disponen de una función para imprimir 
  múltiples juegos de copias. Sin embargo, no utilice esta 
  función si se ha seleccionado un valor distinto de [1 en 1] en
  [Formato de página] en la ficha [Configuración de página].
  Si se utiliza esta función en la aplicación, es posible que 
  los datos no se muestren adecuadamente.

================================================================

3. Separador de página

  [Separador de página] de Windows no puede imprimirse.

================================================================

4. Permisos del usuario en la instalación

  Abra una sesión como usuario con derechos de administrador 
  para instalar el controlador de impresora. Se requieren 
  Permisos de control total para poder realizar la instalación.

================================================================

5. Descarga e instalación de una impresora de red 
   (Windows 2000/XP/Server 2003)

  Cuando descargue e instale una impresora de red en Windows 
  2000/XP/Server 2003, podrá instalarla como usuario sin 
  derechos de administrador.
  Sin embargo, si la impresora de red es instalada por un 
  usuario sin derechos de administrador, es posible que algunas 
  funciones no estén disponibles debido a las restricciones de 
  seguridad del sistema operativo.

  [Funciones no disponibles]
  - El Desinstalador no puede registrarse en [Agregar o quitar 
    programas] ([Agregar o quitar programas] para Windows 2000).

================================================================

6. Instalación en un entorno multiusuario 
   (Windows XP/Vista/7/Server 2008)

  Cuando instale esta impresora en Windows XP/Vista/7/Server 
  2008, no la instale para un usuario seleccionado por la 
  función de "Cambio rápido de usuario" de Windows 
  XP/Vista/7/Server 2008.
  Cuando utilice la función de "Cambio rápido de usuario", 
  asegúrese de iniciar una nueva sesión antes de la instalación.

================================================================

7. Cómo desinstalar la impresora de red conectada por otro 
   usuario

  Aunque ejecute el Desinstalador, la información sobre la 
  impresora de red instalada en el ordenador local de otro 
  usuario no se borrará sino que permanecerá en su sistema 
  operativo como información particular de dicho usuario.
  Si otro usuario ha instalado la impresora en su ordenador 
  local a través de la red, asegúrese de que dicho usuario 
  elimine el icono de la impresora de red de la carpeta de la 
  impresora antes de ejecutar el Desinstalador.

================================================================

8. Limitaciones de la impresora de red

  Cuando se instale una impresora de red, aunque se cambie el 
  destino de la conexión a un puerto local, la impresora no se 
  podrá utilizar como impresora local.
  Si utiliza la impresora como impresora local, desinstale la 
  impresora de red instalada, seleccione un puerto local y, a 
  continuación, instale de nuevo el controlador de impresora.

================================================================

9. Reinicio de Windows con un trabajo de impresión pendiente

  Si necesita reiniciar Windows con un trabajo de impresión 
  pendiente, pause la impresión desde la carpeta de la 
  impresora. A continuación, reinicie Windows.

================================================================

10. Modo protegido de Internet Explorer 
    (Windows Vista/7/Server 2008)

  Si está utilizando Windows Vista/7/Server 2008 y ejecuta 
  Internet Explorer haciendo clic en el icono de [Mostrar 
  manual] etc. en Instalación de CD-ROM, [Modo protegido] se 
  establecerá en [desactivado] independientemente de la 
  configuración.
  Si desea establecer [Modo protegido] en [activado], salga de 
  Internet Explorer y, a continuación, ejecútelo de nuevo 
  manualmente.

================================================================

11. Caracteres para el nombre de la impresora 
    (Windows Vista/7/Server 2008)

  Cuando se utilice un carácter "@" en el nombre de la 
  impresora, el sistema operativo no podrá reconocerla debido a 
  la invalidez del nombre de la misma, inhabilitando de este 
  modo la impresión, el funcionamiento y la configuración.

================================================================

12. Registro de tamaños de papel personalizados 
    (Windows Vista/7/Server 2008)

  Asegúrese de iniciar una sesión como usuario con derechos de 
  administrador cuando edite [Tamaño de papel personalizado] en 
  la ficha [Configuración de página] del controlador de 
  impresora.

================================================================

13. Permiso de acceso a Propiedades de impresora 
    (Windows Vista/7/Server 2008)

  Cuando inicie una sesión como un usuario con derechos de 
  administrador diferente del utilizado para instalar el 
  controlador de impresora, no podrá acceder a las propiedades 
  de la impresora como usuario con derechos de administrador.
  Para acceder a las propiedades de la impresora, lleve a cabo 
  cualquiera de los siguientes procedimientos.

  - Windows Vista y Server 2008
    Haga clic con el botón derecho del ratón en el icono de la 
    impresora y, a continuación, seleccione [Ejecutar como 
    administrador].

  - Windows 7 y Server 2008 R2
    Haga clic en [Cambiar propiedades] en la ficha [General] de 
    las propiedades de la impresora.

================================================================

14. Impresión de líneas finas (Windows Vista/7/Server 2008)

  En ciertas aplicaciones, es posible establecer el grosor de 
  una línea en un valor inferior a 1 punto, pero el grosor real 
  en la impresión podría ser equivalente al de una línea de 1 
  punto.

================================================================

15. Precauciones cuando se utilice Microsoft Excel

  Cuando se impriman datos de Microsoft Excel que incluyan 
  celdas combinadas, los caracteres de los datos podrían no 
  imprimirse.

  El fenómeno anterior se produce al cambiar las opciones de 
  visualización en el cuadro de diálogo [Preferencias 
  (usuarios)] o [Preferencias (administradores)] de la Ventana 
  de estado de impresora, por tanto tenga cuidado cuando cambie 
  las opciones.

  - Ajuste de [Mostrar opciones de tiempo] en [Al iniciar la 
    impresión] (valor predeterminado: [En caso de error]).
  - Ajuste de [Maximizar ventana cuando se muestre 
    automáticamente] en Activado (valor predeterminado: 
    Activado)

================================================================

16. Precauciones a la hora de actualizar de Windows Vista a 
    Windows 7

  Cuando actualice el sistema operativo de Windows Vista a 
  Windows 7, desinstale en primer lugar el controlador de 
  impresora, actualice el sistema operativo y, a continuación, 
  vuelva a instalar el controlador de impresora tras la 
  actualización.
  Si actualiza el sistema operativo sin desinstalar el 
  controlador de impresora, tal vez el controlador no funcione 
  correctamente.
  En este caso, desinstale el controlador de impresora y, a 
  continuación, vuelva a instalarlo.

================================================================

17. Canon Driver Configuration Tool

  Este controlador admite la versión 2.00 o superior de Canon 
  Driver Configuration Tool.
  Puede cambiar la configuración predeterminada de fábrica del 
  controlador de impresora para crear conjuntos personalizados 
  de controladores mediante Canon Driver Configuration Tool.
  Para obtener información detallada acerca de esta utilidad, 
  consulte la Guía del usuario de Canon Driver Configuration 
  Tool.
