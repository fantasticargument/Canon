================================================================

1. Skaalaus

  Jotkin sovellukset eivät ehkä tulosta oikein, jos skaalausarvo
  määritetään. Muuta tällöin skaalausarvoksi 100 %.
  Jos sovelluksessa on oma skaalaustoiminto, määritä 
  skaalausarvo sovelluksessa.

================================================================

2. Sivun asettelu

  Voit tulostaa tiedot useina kopioina joistakin sovelluksista. 
  Älä kuitenkaan käytä tätä toimintoa, kun [Sivun asetukset] 
  -välilehden [Sivun asettelu] -kohtaan on valittu jokin muu 
  asetus kuin [1 sivu arkilla].
  Jos tätä sovelluksessa olevaa toimintoa käytetään, tiedot 
  eivät ehkä tulostu oikein.

================================================================

3. Erotinsivu

  [Erotinsivu] ei voi tulostaa Windowsissa.

================================================================

4. Asennuksen käyttöoikeudet

  Tulostinajurin asentamiseksi kirjaudu käyttäjäksi, jolla on 
  järjestelmänvalvojan oikeudet. Tulostinajurin asennukseen 
  tarvitaan täydet käyttöoikeudet.

================================================================

5. Verkkotulostimen lataaminen ja asentaminen 
   (Windows 2000/XP/Server 2003)

  Kun Windows 2000/XP/Server 2003 -käyttöjärjestelmään ladataan 
  ja asennetaan verkkotulostinta, voit asentaa sen käyttäjänä, 
  jolla ei ole järjestelmänvalvojan oikeuksia.
  Jos verkkotulostin asennetaan ilman järjestelmänvalvojan 
  oikeuksia, kaikki toiminnot eivät ehkä ole käytettävissä 
  käyttöjärjestelmän suojausrajoitusten vuoksi.

  [Ei käytettävissä olevat toiminnot]
  - Asennuksen poisto-ohjelmaa ei voi rekisteröidä [Lisää tai 
    poista sovellus] -toiminnon avulla ([Lisää tai poista 
    sovellus: ominaisuudet] Windows 2000 -järjestelmässä).

================================================================

6. Yhteiskäyttöympäristön asentaminen 
   (Windows XP/Vista/7/Server 2008)

  Jos olet asentamassa tulostimen Windows XP/Vista/7/Server 2008
  -käyttöjärjestelmään, älä vaihda käyttäjää Windows 
  XP/Vista/7/Server 2008 -käyttöjärjestelmän "Nopea aktiivisen 
  käyttäjän vaihto" -toiminnon avulla ennen ohjelman 
  asentamista.
  Käytettäessä toimintoa "Nopea aktiivisen käyttäjän vaihto" 
  muista kirjautua uudestaan sisään ennen asennusta.

================================================================

7. Verkkotulostimen asennuksen poistaminen, kun tulostimella on 
   useita käyttäjiä

  Vaikka suoritat asennuksen poisto-ohjelman, toisen käyttäjän 
  paikallisella tietokoneella olevia verkkotulostimen tietoja ei
  poisteta, vaan ne säilyvät käyttöjärjestelmässä käyttäjän 
  saatavissa.
  Jos toinen käyttäjä on asentanut tulostimen paikalliselle 
  tietokoneelle verkon kautta, varmista, että kyseinen käyttäjä 
  poistaa verkkotulostimen kuvakkeen tulostinkansiosta, ennen 
  kuin asennuksen poistaminen suoritetaan.

================================================================

8. Verkkotulostimen rajoitukset

  Kun verkkotulostin on asennettu, sitä ei voi käyttää 
  paikallisena tulostimena, vaikka muutat yhteyskohteeksi 
  paikallisen portin.
  Jos käytät tulostinta paikallisena tulostimena, poista 
  asennetun verkkotulostimen asennus, valitse paikallinen portti
  ja asenna sitten tulostinajuri uudelleen.

================================================================

9. Windowsin käynnistäminen uudelleen, kun tulostustyö on kesken

  Jos Windows-käyttöjärjestelmä on käynnistettävä uudelleen, kun
  tulostustyö on kesken, keskeytä tulostustyö tulostinkansiosta.
  Käynnistä sitten Windows uudelleen.

================================================================

10. Internet Explorerin suojattu tila 
    (Windows Vista/7/Server 2008)

  Jos käytössä on Windows Vista/7/Server 2008 ja avaat Internet 
  Explorer -selaimen klikkaamalla esimerkiksi [Näytä opas] 
  -kuvaketta CD-ROM-asennuksessa, [Suojattu tila] -asetus on 
  [Poissa käytöstä] asetuksista huolimatta.
  Jos haluat asettaa [Suojattu tila] -asetukseksi [Käytössä], 
  sulje Internet Explorer ja avaa se uudelleen.

================================================================

11. Tulostimen nimen merkit (Windows Vista/7/Server 2008)

  Kun merkkiä "@" käytetään tulostimen nimessä, 
  käyttöjärjestelmä ei tunnista tulostinta virheellisen 
  tulostimen nimen vuoksi, jolloin tulostus, käyttö ja asetukset
  eivät onnistu.

================================================================

12. Mukautettujen paperikokojen rekisteröinti 
    (Windows Vista/7/Server 2008)

  Varmista, että kirjaudut sisään järjestelmänvalvojan 
  oikeuksin, kun muokkaat [Mukautettu paperikoko] -asetusta 
  tulostinajurin [Sivun asetukset] -välilehdellä.

================================================================

13. Tulostimen ominaisuuksien käyttöoikeudet 
    (Windows Vista/7/Server 2008)

  Kun kirjaudut sisään järjestelmänvalvojan käyttöoikeuksin ja 
  jos järjestelmänvalvoja on eri kuin tulostinajurin asentaja, 
  et voi käyttää tulostimen ominaisuuksia järjestelmänvalvojan 
  käyttöoikeuksin.
  Kun haluat päästä tulostimen ominaisuuksiin, tee jompikumpi 
  seuraavista toimenpiteistä.

  - Windows Vista ja Server 2008
    Napsauta tulostimen kuvaketta hiiren kakkospainikkeella ja 
    valitse sitten [Suorita järjestelmänvalvojana].

  - Windows 7 ja Server 2008 R2
    Napsauta [Muokkaa ominaisuuksia] tulostimen ominaisuuksien 
    [Yleiset] -välilehdellä.

================================================================

14. Ohuiden viivojen tulostus (Windows Vista/7/Server 2008)

  Tietyissä sovelluksissa viivan paksuus voidaan asettaa 
  pienemmäksi kuin 1 pt, mutta viivan todellinen paksuus 
  tulosteessa saattaa vastata 1 pt:n viivaa.

================================================================

15. Huomautuksia Microsoft Excel -ohjelman käytölle

  Tulostaessasi yhdistettyjä soluja sisältävää Microsoft Excel 
  –tiedostoa tietojen sisältämät merkit eivät ehkä tulostu.

  Koska edellä mainittu ilmiö esiintyy näyttöasetusten vaihdon 
  yhteydessä tulostimen tilaikkunan valintaruudussa [Määritykset
  (käyttäjät)] tai [Määritykset (järjestelmänvalvojat)] 
  seuraavaksi kuvatulla tavalla, ole varovainen muuttaessasi 
  asetuksia.

  - Asetus [Näytä ajoitusasetukset] valintaan [Tulostusta 
    aloitettaessa] (oletus: [Virheen ilmetessä])
  - Asetus [Suurenna ikkuna näytettäessä automaattisesti] 
    valintaan Käytössä (oletus: Käytössä)

================================================================

16. Huomioita päivitettäessä Windows Vistasta Windows 7 
    -käyttöjärjestelmään

  Kun päivität käyttöjärjestelmän Windows Vistasta Windows 
  7:ään, poista ensin tulostinohjaimen asennus, päivitä 
  käyttöjärjestelmä ja asenna tulostinohjain sitten uudelleen 
  päivityksen jälkeen.
  Jos päivität käyttöjärjestelmän poistamatta tulostinohjaimen 
  asennusta, tulostinohjain ei saata toimia kunnolla.
  Poista tällaisessa tapauksessa tulostinohjaimen asennus kerran
  ja asenna tulostinohjain sitten uudelleen.

================================================================

17. Canon Driver Configuration Tool

  Tämä ajuri tukee Canon Driver Configuration Tool -ohjelman 
  versiota 2.00 tai uudempaa.
  Voit muuttaa tulostinajurin tehdasoletusasetuksia ja luoda 
  mukautettuja ajurisarjoja käyttämällä Canon Driver 
  Configuration Tool -ohjelmaa.
  Tarkkoja tietoja tästä apuohjelmasta on Canon Driver 
  Configuration Tool -ohjelman User's Guide -käyttöoppaassa.
